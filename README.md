# Smart Mirror

## 🚀 About Us
해당 프로젝트는 대학 졸업작품으로, IoT와 AI를 결합한 일상에 의미있는 작품을 목표로 개발된 프로젝트입니다.

일상 생활에서 빠질 수 없는 `거울`과 `AI`를 결합하여, 현재 내 상태의 피부를 진단하는 `스마트 미러`를 만들어보자는 취지에서 시작되었습니다.

- 인원 : 4명
- 역할
    - 피부진단 학습 모델 개발
    - 팀장 / 프로젝트 매니지먼트 담당

## Projects
- smart_mirror_fe - 프론트엔드
    - 입력장치는 `카메라`만을 사용하는 Web 서비스입니다
- smart_mirror_be - 백엔드
    - 모델 학습과 API 서버로 활용되었습니다
- data_view - QR 코드로 데이터를 사용자 휴대폰으로 받아볼 수 있는 사이트
    - 해당 코드는 오로지 URL 리소스를 받아 출력하는 목적으로 개발되었습니다
    - 이는 더 좋은 서비스로 만들기 위해 추가된 기능으로, AI로 제작하였습니다.

## ISSUE
  - 딥러닝 지식 부족!
    - 팀원들의 도움과 깃허브 및 많은 블로그를 찾아보며 학습
  - 학습 데이터 부족 및 라벨링 작업의 어려움
    - 사진 데이터를 수집하는 것과 처음 해보는 라벨링 작업으로 인해 많은 시간을 소비.
    - chatgpt를 통해 라벨링하는 방법 및 분류 기준을 상세화하였고, 피부 상태 심각도를 직접 판단하여 라벨링을 진행.


## Review
이 프로젝트로 대학 내에서 3위로 수상하였지만, 세부적으로 검토한다면 치명적인 점이 많을 것으로 생각합니다. 많은 시간을 들여서 밤을 새워가면서까지 프로젝트를 진행하였지만 제출까지의 남은 기간의 압박으로 인해 내 것이 아닌 chatgpt에서 생성한 코드들로 채워 프로젝트를 완성하였습니다.

비록 내부적으로 뛰어난 평가를 받을 수 있는 코드는 아니지만 밤을 세워가며 팀원 전체의 노력이 담겨 있는 프로젝트였기에 유의미한 경험이었다고 생각합니다.

졸업 후 다른 진로의 길을 가기로 한 팀원이 2명이나 있었지만 끝까지 포기하지 않고 참여해준 팀원들에게 감사하며 특별히 인프라 및 프론트엔드 전체를 담당하며 가장 많은 기여를 한 팀원에게 너무 감사했습니다. 저도 팀원들에게 도움이 되고 의지가 되는 그런 개발자가 되겠습니다!!!
